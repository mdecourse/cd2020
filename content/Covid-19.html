Covid-19

School post.pdf

為確保遠距教學之授課品質與學習成效，請教師於109 年4 月24日之前先行演練同步或非同步遠距教學，並於演練後，請將演練實況紀錄上傳eCampus(或上傳雲端硬碟並將連結點發布至eCampus)作為影音教材，並請於線上填寫演練紀錄與回饋單(https://forms.gle/CoTQaE3Hs31yzUX37)。

Course arrangement.pdf

Since we do not use eCampus but the Github, Gitter and Youtube, therefore we will use Youtube + Portable OBS as online course tool. And we will use https://meet.google.com to meet with each group leader online to give assignment for each group.

Online course process:

EMail https://meet.google.com meeting url to each group leader to conduct online meeting.

Instructor use Youtube and OBS to stream weekly course lecture and post its live URL on class Gitter.

All the students use Youtube and OBS to stream weekly assignment and post its live URL on each group's website.

All the students complete self-assessment and peer-assessment process according to the results on each member's Github commits, Github Page website and Youtube video.

線上課程流程：

通過電子郵件將 https://meet.google.com 會議網址發送給各組組長, 以進行線上會議.

講師使用 Youtube 和 OBS 直播每週的課程內容, 並將其 URL 發佈到各班 Gitter.

所有學員使用 Youtube 和 OBS 直播並錄製當週作業, 並將其 URL 發佈到各分組的網站上.

所有學員根據各學員的 Github 提交, Github Page 網站與 Youtube 影片內容, 完成自評與互評.
